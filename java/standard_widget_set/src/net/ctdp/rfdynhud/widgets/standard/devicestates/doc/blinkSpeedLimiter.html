The blinkSpeedLimiter property defines, whether the speed limiter icon will be forced visible and blink, if it is disabled, but should be enabled or vice versa.
The following conditions must be met.

  • If the speed limiter device is installed for the current vehicle
  • If the speedLimiterVisibility property is not set to Never
  • One of:
    • If the vehicle is in the pitlane AND the speed limiter is currently disabled
    • If the vehicle is NOT in the pitlane AND the speed limiter is currently enabled

Note, that this property may override the speedLimiterVisibility property, if it is not set to Always or Never.
