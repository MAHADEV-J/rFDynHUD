The ignitionVisibility defines, when the ignition icon is visible and when it is hidden.

  • Never: This icon for this device will always be hidden.
  • Always: This icon for this device will always be displayed regardless of its state.
  • If not Off: This icon for this device will be shown, if and only if the device is NOT switched off.
  • If not On: This icon for this device will be shown, if and only if the device is NOT switched on.
